About Threat Feeds

All four abuse.ch feeds are completely free and require no API keys:

  • URLhaus - Malicious URLs
  • ThreatFox - IOCs shared by the community
  • Feodo Tracker - Botnet C2 infrastructure
  • MalwareBazaar - Malware samples and hashes

API keys are only required for IOC enrichment (VirusTotal, AbuseIPDB) and AI briefings (Gemini).
